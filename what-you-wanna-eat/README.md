# what you wanna eat?

A dependency-free mini web app for deciding what to eat. Choose Brunch / Breakfast, Main Course, or Dessert, then spin the wheel.

## Run locally

```bash
npm run dev
```

Open `http://127.0.0.1:4173` in a browser.

The app is also a plain static site, so any local static-file server will work.

## Included options

- Brunch / Breakfast: 34
- Main Course: 119
- Dessert: 42

All copy is in English. The interface uses Times New Roman and custom sketch-style food illustrations.
